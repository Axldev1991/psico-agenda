# Funcionalidad: Gestión de Pacientes
**Estado:** Definición Conceptual

---

## 1. Objetivo
Permitir a la profesional administrar la base de datos de sus pacientes de forma centralizada y segura.

## 2. Requerimientos
*   **Alta de Paciente:** Registro de Nombre completo, DNI/Identificación, Teléfono de contacto, Email y Obra Social/Prepaga.
*   **Detección de Modalidad:** Definir si el paciente es atendido siempre de forma **Presencial** o **Virtual**.
*   **Ficha Clínica:** Un panel donde se visualicen los datos del paciente y se tenga acceso rápido a su historial y próximos turnos.
*   **Estado del Paciente:** Poder marcar pacientes como "Activos" o "De alta/Inactivos" para limpiar la vista principal.
