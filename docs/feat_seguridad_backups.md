# Funcionalidad: Seguridad y Backups
**Estado:** Definición Conceptual

---

## 1. Objetivo
Garantizar la privacidad absoluta de los datos y su disponibilidad ante cualquier fallo técnico.

## 2. Requerimientos
*   **Autenticación:** Uso de Google OAuth 2.0 (Login con Google).
*   **Soberanía de Datos:** Almacenamiento en espacio del usuario (Drive).
*   **Backup a Disco Externo:** Función para "Descargar todo el Proyecto" en un archivo comprimido (.zip) estructurado.
*   **Privacidad:** Los datos clínicos son legibles por archivos individuales para que el usuario no dependa nunca de la App.
